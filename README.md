# SRT Translator

이 프로젝트는 DeepL API를 사용하여 SRT 파일 내의 자막을 번역하는 파이썬 스크립트입니다. Tkinter를 사용하여 파일 선택과 저장을 위한 GUI를 제공합니다.

## 기능

- SRT 파일 내의 자막을 번역
- 번역된 SRT 파일을 지정된 폴더에 저장
- DeepL API를 사용한 번역

## 요구 사항

- Python 3.x
- `requests` 라이브러리
- `tqdm` 라이브러리
- `tkinter` 라이브러리 (대부분의 Python 배포판에 기본 포함)
- DeepL API 키
